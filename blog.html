Answers to DOM Questions

Question 1: What are the different ways to select an element in the DOM?

WE can select elements in the DOM using the following methods:

  • getElementById: Selects an element by ID.
  • getElementsByClassName: Selects all elements with a specified class.
  • getElementsByTagName: Selects all elements with a specified tag name.
  • querySelector: Selects the first element that matches a CSS selector.

    • etc.

Question 2: What is the difference between innerHTML, innerText, and textContent?

  • innerHTML: Gets or sets the HTML content of an element. It can include HTML tags.
  • innerText: Gets or sets the text content of an element, excluding any HTML tags, and reflects the visible text.
  • textContent: Similar to innerText but is more consistent. It includes all text content in the element, including hidden text.

Question 3: What is event delegation in the DOM?

Event delegation is a technique where we can attach a single event listener to a parent element instead of adding event listeners to each child element. The event listener captures events from child elements through the event bubbling process.

Question 4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts at the target element and then bubbles up to the parent elements, triggering the event listeners on each parent element.

Question 5: How do you create, add, and remove elements using JavaScript?

You can use the following methods:

  • createElement: Creates a new element.
  • appendChild: Adds a new element as a child to another element.
  • removeChild: Removes a child element from its parent.
